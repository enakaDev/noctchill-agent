# コミュデータ管理ガイド

このディレクトリには、各アイドルの人格形成に使用する原作コミュの文字起こしデータを格納します。

## ディレクトリ構造

```
commu/
├── README.md           # このファイル
├── noctchill/          # ノクチル全員が登場するコミュ
├── asakura/            # 浅倉 透 メインのコミュ
├── higuchi/            # 樋口 円香 メインのコミュ
├── fukumaru/           # 福丸 小糸 メインのコミュ
└── ichikawa/           # 市川 雛菜 メインのコミュ
```

## 対応ファイル形式

- `.md` - Markdown形式（推奨）
- `.txt` - プレーンテキスト形式

どちらの形式でもコミュデータを追加できます。

## コミュファイルの追加方法

### 1. ファイル名の命名規則

```
{番号}_{コミュ名}.md
{番号}_{コミュ名}.txt
```

例：
- `001_天塵.md`
- `002_薄桃色にこんがらがって.txt`
- `003_いつもの放課後.md`

### 2. ファイルの配置場所

| コミュの種類 | 配置先 |
|-------------|--------|
| ノクチル全員が登場 | `commu/noctchill/` |
| 浅倉透がメイン | `commu/asakura/` |
| 樋口円香がメイン | `commu/higuchi/` |
| 福丸小糸がメイン | `commu/fukumaru/` |
| 市川雛菜がメイン | `commu/ichikawa/` |

### 3. ファイル内容のフォーマット

各ディレクトリ内の `template.md` を参考にしてください。

基本構造：

```markdown
# コミュタイトル

## 基本情報
- 種類: メインストーリー / サポート / イベント
- 登場キャラ: 透、円香、小糸、雛菜

## 文字起こし

透「セリフ」

円香「セリフ」

（ナレーション）

## このコミュから読み取れる性格特性
- ポイント1
- ポイント2
```

### 4. プレーンテキスト形式（.txt）の場合

Markdown記法を使わなくても問題ありません。

```
【コミュタイトル】

[基本情報]
種類: メインストーリー
登場キャラ: 透、円香、小糸、雛菜

[文字起こし]

透「セリフ」

円香「セリフ」

（ナレーション）

[性格特性]
- ポイント1
- ポイント2
```

## 性格分析まとめファイル

各アイドルのディレクトリには `personality.md` を作成できます。
複数のコミュから読み取れる性格特性を集約したサマリーとして活用してください。

## 文字起こしデータの利用について

**重要**: コミュの文字起こしは精度が悪く、日本語として不自然な箇所が含まれる可能性があります。

エージェントがコミュデータを参照する際は：
- **日本語として意味のある箇所のみを参考にする**
- 意味不明な文章や文字化けは無視する
- 文脈から判断して自然な解釈を心がける
